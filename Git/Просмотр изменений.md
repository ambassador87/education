### git diff
Показывает изменения  между двумя коммитами
```bash
# Показывает изменения с поледнего коммита, которые еще не были добавлены в индекс
git diff

# Показывает изменения добавленные в индекс
git diff --staged
git diff --cached

# Показывает любые изменения с последнего коммита, даже если они добавлены в индекс
git diff HEAD

# Показывает изменения в файлах в немного более удобном виде 
git diff --staged --word-diff

# Не показывает в изменениях пустые строки
gid diff --cached -w

```
Чтобы посмотреть изменения между двумя коммитами:
```bash
# Показывает изменения между предпоследним и последним коммитом
git diff HEAD~1

# Показывает изменения между указанным и последним коммитом
git diff 10f744

# Показывает изменения между двумя коммитами
git diff aad7be 10f774

# Показывает изменения для конкретного файла
git diff aad7be web_pages/index.html
```
`git diff` позволяет сравнивать два файла, даже если они не находятся в репозитории.
Подымемся на уровень выше каталога с репозиторием и сравним два файла:
```bash
git diff page.html index.html
```
Покажет отличия красным и сходства зеленым цветом

### git show
Показывает изменения в указанном коммите, в том время как `git diff` показывает накопленную разницу между двумя коммитами. 
Но git show так же может показывать разницу между двумя коммитами.
```bash
git show 0b7ff2..43d3f
```
В этом случае вывод git diff и git show будет очень похож.

Чтобы посмотреть какие файлы были изменены в указанном коммите:
```bash
# Покажет только имена файлов, которые затронул этот коммит
git show 4b22e8 --name-only
# page1.html

# Покажет имена файлов, затронутые с указанного и до последнего коммита
git diff 4b22e8 --name-only
# css/styles.css
# page1.html
# web_pages/index.html
# web_pages/page2.html
```
