Как выглядит переименование файла в терминале :
```bash
mv page.html index.html
git status
# покажет 
# deleted: page.html
# untracked files: index.html
```
Теперь для того, чтобы зафиксировать изменение в индексе нужно сделать еще два действия:
```bash
git add index.html
git rm page.html
```

Или сделать это в одно действие
```bash
git mv page.html index.html
git status
# renamed: page.html -> index.html
```
Файл будет переименован и добавлен в индекс с новым именем
В Linux переименование и перемещение файла это одно и то же. Поэтому сделать перемещение файла в другой каталог можно сделать той же командой
```bash
git mv style.css static/styles
```
То же самое работает и для директорий. Так же для переноса можно воспользоваться шаблонами. Например, перенести все html-файлы в каталог web_pages:
```bash
git mv *.html web_pages/
```
