## git blame
Команда для поиска автора каждой строки.

Чтобы посмотреть кто, когда и что поменял в конкретном файле:
```shell
git blame web_pages/index.html
```
Допустим мы нашли в файле строки, которые вызывают у нас вопросы. Мы можем узнать кто автор этих  
строк, чтобы связаться с ним:
```shell
# Показывает email автора вместо имени
git blame web_pages/index.html -e

# Показывает информацию о конкретных строках
git blame index.html -L 8,10 -e

# Показывает инфо с 8 строки и до конца файла
git blame index.html -L 8 -e

# Показывает только инфо одной 8 строки
# -w игнорирует коммиты, в которых менялось только форматирование (пробелы, отступы)
git blame index.html -L 8,8 -e -w

# Фильтрует изменения по датам
git blame --since="3 weeks ago" index.html
git blame --since="2025.01.01" index.html
```
