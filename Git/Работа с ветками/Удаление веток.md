Есть 3 ситуации, когда мы хотим удалить ветку:
* Ветка пустая (без коммитов)
* Ветка не пустая и было слияние с мастер веткой
* Ветка не пустая и не было слияния с мастер веткой

В зависимости от этого ветки будут удаляться по разному:
```bash
# Если ветка не пустая и был merge с мастер веткой 
# или ветка пустая, то её можно просто удалить
git branch -d <имя_ветки>

# Если ветка не пустая, но не слита в мастер ветку,
# просто удалить её не получится, нужно либо слить её в мастер
# либо использовать ключ -D
git branch -D <имя_ветки>
```

Для переименования ветки используется ключ -m (move):
```bash
git branch -m master main
```

Поиск веток по подстроке в имени
```bash
# Ищет все ветки, у которых в названии есть слово 'feature'
git branch --list '*feature*'
```
