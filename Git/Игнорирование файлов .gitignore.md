Если мы не хотим чтобы Git видел какие-то файлы, и не предлагал нам добавить их в индекс, мы можем указать их в .gitignore по имени или по шаблону
```
# Шаблон для всех log-файлов
*.log

# Шаблон для каталогов и Git будет игнорировать файлы в папке logs
logs/*

# Шаблон для файла исключения, который нужно отслеживать
!critical_logs.log
```

Если нужно отслеживать файл, несмотря на то, что он подходит под шаблон в .gitignore, то его можно добавить в индекс принудительно:
```bash
# Ключ -f и имя файла принудительно добавят файл в индекс
git add -f debug.log
```

```
## глобальные настройки
# указываем глобальную настройку для пути к файлу .gitignore
git config --global core.excludesfile ~/.gitignore_global
# создаем сам файл по указанному пути
touch ~/.gitignore_global
nano ~/.gitignore_global
```
